# Ex03: 使用 millis() 控制SOS闪烁信号

## 实验目标
使用 `millis()` 函数实现非阻塞式 SOS 摩尔斯电码闪烁信号。

## SOS信号定义 (摩尔斯电码)
```
S: ● ● ●   (三短: 亮200ms/灭200ms x3)
O: — — —   (三长: 亮600ms/灭200ms x3)  
S: ● ● ●   (三短: 亮200ms/灭200ms x3)
```

## 时间参数
| 参数 | 值 | 说明 |
|------|-----|------|
| DOT_ON | 200ms | 短闪-点亮 |
| DOT_OFF | 200ms | 短闪-熄灭(符号间隔) |
| DASH_ON | 600ms | 长闪-点亮 |
| DASH_OFF | 200ms | 长闪-熄灭(符号间隔) |
| LETTER_GAP | 500ms | 字母间隔(S→O, O→S) |
| WORD_GAP | 2000ms | 单词间隔(SOS结束后) |

## 设计方法: 状态机 (State Machine)
将整个SOS信号分解为20个步骤（每步包含"亮"和"灭"两个阶段）：
- 步骤 0~5: 第一个 S (● ● ●)
- 步骤 6~11: O (— — —)
- 步骤 12~17: 第二个 S (● ● ●)
- 步骤 17之后: 单词间隔，然后循环

每个步骤用 `millis()` 判断时间到达后切换到下一步。

## 核心优势
- ✅ 完全非阻塞（不使用delay）
- ✅ 精确的时序控制
- ✅ 可扩展（可同时处理其他任务如按键检测、WiFi通信等）
